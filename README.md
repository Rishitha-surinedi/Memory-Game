# Emoji Memory Game

A fun and interactive browser-based memory matching game built using HTML, CSS, and JavaScript.

## Live Demo

🔗 https://rishitha-surinedi.github.io/Memory-Game/

## Features

* Emoji card matching gameplay
* Move counter
* Timer tracking
* Restart game functionality
* Confetti celebration on completion
* Responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Play

1. Click on a card to reveal an emoji.
2. Match pairs of identical emojis.
3. Complete all matches with the fewest moves and shortest time.
